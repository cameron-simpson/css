histbackup and histbackup-prune

Last modified: Jul 18 17:06

Overview

histbackup makes datestamped snapshots of a directory (possibly remote) with incremental cost for each extra snapshot.

See also: the manual histbackup.1.html.

histbackup-prune removes older snapshots from the archive if the free space in the filesystem is below a threshold, keeping a certain minimum number of snapshots regardless.

Uses rsync [http://freshmeat.net/projects/rsync/"].
