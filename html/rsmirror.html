rsmirror

Last modified: Apr 11 09:19

Overview

rsmirror maintains a mirror of a set of rsync repositories.

Also see the manual entry: rsmirror.1.html.

Installation

It's just a script - put it in your command $PATH. It requires rsync(1) for rsync repositories and wget(1) for http repositories. If you use the -t timeout option you will need my timeout script as well.
